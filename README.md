Portfolio 2025

Dieses Projekt ist eine Portfolio-Website mit separatem Frontend und Backend.

Struktur

- `frontend`: UI, statische Daten und Styles
- `backend`: API für den KI-Assistenten

Frontend (lokal)

1. Abhängigkeiten installieren: `npm install`
2. Entwicklung starten: `npm run dev`

Frontend Build

- Produktions-Build: `npm run build`
- Start (Production): `npm run start`
- Lint: `npm run lint`

Backend (lokal)

1. Virtuelle Umgebung anlegen (optional): `python -m venv .venv`
2. Abhängigkeiten installieren: `pip install -r backend/requirements.txt`
3. Backend starten: `python backend/main.py`
